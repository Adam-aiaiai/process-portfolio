Core Positioning

Go Runners is an exploratory gamified web solution that bridges the gap between digital motivation and physical onsite running. We aim to transform "isolated exercise" into a playful social journey through location-based interactions, adhering to the principles of human-centric design.

Research Directions

The Problem: Boredom and lack of community engagement in solo running; high organizational friction for local run-event hosts.
Proposed Solution: A lightweight web app featuring location-specific missions, real-time leaderboard synchronization, and an intuitive organizer dashboard.
Constraints & Context: Mobile-first architecture, deployed via GitHub Pages, prioritizing low-friction user onboarding.
